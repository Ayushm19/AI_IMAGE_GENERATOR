### AI Image genrator
Demo Link : https://aiimgen.netlify.app/
### setup .env file
```js
MONGODB_URL=...
OPENAI_API_KEY=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```

After installing all the dependencies
run this command for both client and server 
```shell
npm start
```
